# Iris Flower Classification

## Objective
Train a machine learning model to classify iris flowers into 3 species: Setosa, Versicolor, and Virginica based on physical measurements.

## Dataset
Built-in sklearn dataset — no download required.

## Tools & Libraries
Python, Jupyter Notebook, pandas, numpy, matplotlib, seaborn, scikit-learn

## Models Used
- Logistic Regression
- K-Nearest Neighbours (KNN)

## Results
Both models evaluated using accuracy score, confusion matrix, and classification report. Best model identified with justification.

## How to Run
1. Open `task1.ipynb` in Jupyter Notebook or VS Code
2. Run all cells in order
